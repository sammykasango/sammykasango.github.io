Case Study

A deep-dive into a real-world engineering challenge and how it was solved.

Featured Case Study

WiFi Billing & Hotspot Management Platform

01 The Problem

A small ISP provider in Nairobi was managing hotspot billing manually — tracking active sessions in spreadsheets, issuing vouchers by hand, and reconciling payments without any automation. Revenue was leaking, customer experience was poor, and the process was completely unscalable.

02 The Approach

I designed and deployed a self-contained billing system integrated with MikroTik RouterOS. The architecture separated the network authentication layer (RADIUS) from the billing frontend, allowing clean session management, real-time monitoring, and automated voucher generation.

03 Architecture

The system included: a Python-based admin panel, MikroTik API integration for session control, a MySQL backend for user and transaction records, and a simple customer-facing captive portal. Deployed on a local server with remote management capability.

04 Results

100% Billing Automated
60% Admin Time Saved
0 Revenue Leakage
